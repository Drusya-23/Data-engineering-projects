In this bootcamp project I have 5 financial data csv files in my on-premise system. I want to load my files to cloud in ADLS gen2 as bronze layer is my first step for this I use a self hosted runtime in azure data factory. Then I clean the data using transformations in the dataflow as another pipeline and convert and store it as delta files in my adls silver container. Now my intention is to perform SCD Type 1 and 2 transformatipns on the files using dataflows as my third pieline activity. Now I connect these pipeline in my master pipeline using execute pipeline activity and trigger it to run on specific times. Finally by successfully running the pipeline loaded my data in the SQL database.
