# Data-engineering-projects
In this project the main aim is to familarize with the Fabric UI for data loading and transformations. In this project I am incrementally loading some csv files from my on-premise to fabric using on-prem gateway connection. Then I load my data using a copy activity into the lakehouse tables which is my bronze layer. Then I clean the data using the data flows and load them to warehouse tables. That is my silver layer. Next I performed SCD Type 1 transformations on the files and load them to warehouse with the help of Fabric notebooks. I also connect all these together using a master pipeline and schedule it and also set an email notification on successful running of the pipeline.
