In this project I have 5 csv files related to financial details of customers in my storage account. My aim is to transform the files using databricks notebook. For that I created a new connection with my ADLS storage account and Data bricks notebook using the service principal method. Then I performed cleaning transformations on the files and loaded them to storage account as Parquet files. Then I created a new Parquet file using the join transformation to create a new file with specific columns. Now I used this file to create a visualization using power BI and Fabric. I also perform SCD Type 1 transformation on the 5 Parquet file and load them in the gold layer.
